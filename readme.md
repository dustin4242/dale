# Dustin's Awesome Line Editor
Temporary ReadMe before I actually make a flushed-out one.

Simple Installation:
```shell
cargo install dale
```

Simple Usage:
```shell
dale example_file.txt
```

Simple Keybinds:
`Ctrl-S` to save the file
`Ctrl-R` to run a command within the editor
`ESC` to get out of the editor
